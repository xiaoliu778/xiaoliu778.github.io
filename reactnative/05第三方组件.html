WebView

  • 安装
    • yarn add react-native-webview@12.0.2
  • 配置
    • https://github.com/react-native-webview/react-native-webview
  • 使用
    • 直接指定uri地址
    • 直接渲染html代码

Picker(下拉框) 暂且不能用

  • 安装
    • npm install @react-native-picker/picker
  • 配置
    • https://github.com/react-native-picker/picker
    • 注意：不同版本的配置方式不同
  • 使用
    • 注意平台之间的差异 (Android | IOS )

一个react-native下拉/选择器/选择器组件(优先+1)

  • 安装
    • npm install react-native-modal-dropdown -save
  • 配置
    • https://github.com/sohobloo/react-native-modal-dropdown
  • 使用
    • 注意平台之间的差异 (Android | IOS )

Swiper (轮播效果)

  • 安装
    • npm install react-native-swiper
  • 使用
    • https://github.com/leecade/react-native-swiper
  • 注意
  • Swiper必须放在ScrollView中才会有效果

AsyncStorage

  • 安装
    • npm install @react-native-async-storage/async-storage
  • 配置
    • https://github.com/react-native-async-storage/async-storage
  • 使用
    • 增删改查

封装AsyncStorage

  • 增
    • set(key,value)
  • 删
    • delete(key) |clear()
  • 改
    • update(key,value)
  • 查
    • get(key)

Geolocation(获取当前位置)

  • 安装
    • npm install @react-native-community/geolocation
  • 配置
    • https://github.com/michalchudziak/react-native-geolocation
    • 添加获取位置信息的授权许可
  • 使用
    • 通过手机，获取经纬度信息

Camera (摄像头)老版不兼容

  • 安装
    • npm i react-native-camera -S
  • 配置
    • https://github.com/react-native-camera/react-native-camera
    • 添加使用摄像头的授权许可
  • 使用
    • 拍照、扫码、人脸识别……

Camera (摄像头)新版不好用

  • 安装
    • npm i react-native-vision-camera
  • 配置
    • https://github.com/mrousavy/react-native-vision-camera
  • 使用
    • 拍照、扫码、人脸识别、

React-native-image-picker(这玩意好用)

  • 安装
    • npm install react-native-image-picker
  • 配置
    • https://github.com/react-native-image-picker/react-native-image-picker
    • 跟Camera一致，不知与新版一致不
  • 使用
    • 调用摄像头
    • 访问相册
    • 在 android/gradle.properties中设置 newArchEnabled属性 为 true
  • 依赖
  • yarn add react-native-image-picker
  • react-native link react-native-image-picker
