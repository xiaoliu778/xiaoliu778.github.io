简介

  • RN中的路由是通过React-Navigation来完成的
    • React中通过React-Router实现路由
    • RN.044之前，React-Navigation在核心中维护，0.44之后，独立维护
  • 目前使用React-Navigation 5.x版本
    • 官网：https://reactnavigation.org/

基础组件

  • 安装
    • npm install @react-navigation/native
    • npm install react-native-reanimated react-native-gesture-handler react-native-screens react-native-safe-area-context @react-native-community/masked-view
      • react-native-reanimated(动画增强组件)
      • react-native-gesture-handler(手势处理组件)
      • react-native-screens(屏幕处理组件)
      • react-native-safe-area-context(处理刘海屏，针对于安全区域做处理的)
  • 连接
    • RN 0.60后安卓环境自动链接路由 (Android无需任何操作)
    • iOS 下需要手动链接路由 ( npx pod-install ios)
  • 添加头部
    • 将如下代码，放到应用的头部 ( 例如：放到index.js或App.js文件的头部 )
    • import ‘react-native-gesture-handler’
  • 添加导航容器
    • 我们需要在入口文件中，把整个应用，包裹在导航容器 (NavigationContainer) 中 (例如：在index.js或App.js文件中)

Stack导航

  • 简介
    • RN中默认没有类似浏览器的history对象
    • 在RN中跳转之前，先将路由声明在Stack中
  • 安装
    • npm install @react-navigation/stack
  • 使用
    • import [ createStackNavigator ] from ‘@react-navigation/stack’
    • const Stack = createStackNavigator();

Navigator属性

  • initialRouteName
    • 初始化路由，即默认加载的路由
  • headerMode
    • float: iOS头部效果
    • screen: Android 头部效果
    • none：不显示头部
  • screenOptions 0.6版本以后不在支持并且改到了options中

Screen属性

  • options
    • title (标题)
    • headerTitleStyle (标题样式)
    • headerStyle (页头样式)
    • headerLeft (左侧内容)
    • headerRight (右侧内容)
    • headerTintColor (标题颜色)、

React-native-vector-icons (图标组件库)

  • 安装
    • npm install –save react-native-vector-icons
  • 将图标连接到应用 (环境问题较多)
    • https://github.com/oblabor/react-native-vector-icons
  • 使用
    • 需要到具体的图标库官网查看
    • 例如：lonicons、FontAwesome、AntDesign

Drawer导航

  • 安装
    • npm install @react-navigation/drawer
    • 使用Drawer是需要一些额外的库
      • https://docs.swmansion.com/react-native-gesture-handler/docs/installation
        • npm install –save react-native-gesture-handler
      • https://docs.swmansion.com/react-native-reanimated/docs/fundamentals/installation/
        • npm install react-native-reanimated
  • 使用
    • import {createDrawerNavigator} from ‘@react-navigation/drawer’;
    • const Drawer = createDrawerNavigator();
    • 声明 Drawer.Navigator 和 Drawer.Screen

配置Drawer导航

  • Navigator属性
    • drawerPosition：菜单显示位置，left(默认) | right
    • drawertype：菜单动画效果，front | back | slide | permanent
    • drawerStyle: 菜单样式
      • backgroundColor、width…
    • drawerContentOptions：选中菜单项样式
      • activeTintColor：当前选中菜单字体颜色
      • itemStyle：所有菜单项样式
  • Screen属性
    • options
      • title：菜单标题
      • drawerLabel：代替title，返回复杂的组件。{focused:boolean,color:string}
      • drawerIcon: 返回图标的函数 {focused:boolean,color:string,size:number}
      • headerShown：是否显示header。布尔值，默认false不显示
      • headerLeft：函数，声明header左侧的显示内容
      • headerRight:函数，声明header右侧的显示内容
