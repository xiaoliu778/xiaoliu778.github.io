Anaconda修改python版本

  • 查看选择修改版本号
    • conda search –full –name python
  • 使用python -V命令查看当前虚拟环境的python版本：
    • python -V
  • 已知python版本为3.8.0，现在我想把它升级为3.10.6。
    • 使用命令：
      • conda install python=3.10.6 //3.8版本号
    • 如果在conda install python=3.10.6中遇到问题，例如：
    • 则可以先使用：
      • conda uninstall python
      • 对前版本python进行卸载，然后再使用conda install python=3.10.6安装即可。
