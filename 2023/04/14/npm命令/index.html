npm install –global、–save、 –save-dev的区别和使用场景

npm install(i) moduleName

  • 安装到本项目node_modules目录下

npm install –global(-g)

  • 全局安装
  • 是一些需要通用的工具类安装，比如npm、yarn

npm install –save(-S)

  • 是项目需要的，是项目运行必备，如：vue、element-ui等

npm install –save-dev(-D)

  • 是开发时需要的，但是项目运行不必须的，不需要打包到项目里的，如：webkpack、gulp等
